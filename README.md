# ✅ To-Do App

A modern and responsive task management application built using HTML, CSS, and JavaScript. This app allows users to efficiently manage daily tasks with persistent data storage.

## 🚀 Live Demo
👉 (https://dashing-crepe-c96593.netlify.app/)

## 🛠️ Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clear all tasks
- Data saved using localStorage (tasks remain after page reload)

## 🧠 Technologies Used
- HTML
- CSS
- JavaScript

## 💡 How It Works
Tasks are stored in the browser using localStorage, allowing users to keep their task list even after refreshing or closing the browser.
